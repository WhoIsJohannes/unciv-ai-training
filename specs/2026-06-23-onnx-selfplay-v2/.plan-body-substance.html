Locked decisions (the design IS the deliverable)

Spatial encoder = permutation-invariant masked tile-set pool (NOT grid-CNN)
Per-tile (col,row) is NOT recoverable from shard data (zeroBasedIndex is insertion order; positions not emitted; maps hexagonal) and "no new emitted data" is a hard non-goal. Rejected: grid-CNN. No positional feature (insertion-order ≠ 2D position). Confidence: high.
Actor-critic + GAE over ordered trajectories, reward terminal-only
δ_t=r_t+γV(s_{t+1})−V(s_t), V(terminal)=0; A_t=Σ(γλ)^l δ_{t+l}; R_t=A_t+V(s_t); γ=0.99,λ=0.95. Dataset keeps ALL ordered non-terminal learner steps (no-action steps retained for value/GAE; policy-grad masked to acting heads). Rejected: v1 reward-broadcast (breaks GAE). Confidence: high.
Default plain A2C+GAE single-epoch for the attributable curves
PPO-clip would conflate "critic" with "PPO" in the isolated comparison. PPO (ε=0.2,K=4) is an optional non-default knob. Confidence: high.
Value head is training-only; play ONNX is policy-only
export drops the value head; negative test asserts no value output. Confidence: high.
Acceptance = run the experiment correctly + report honestly
p<0.05 is the hypothesis under test, not a ship gate (brief AC3 permits "state plainly if not"). Bounded by a hard compute budget + checkpoint/resume. Confidence: high.
